# 8bit-midi-player
8bitでmidiを再生できます。

プレイはこちらから play here


音域などからトラックを自動で分けます。

BPMも設定できます。
これを使用して動画を作ってもいいですが、自作宣言はやめてください。

You can play MIDI files in 8-bit.  

Tracks are automatically separated based on things like the pitch range.  

You can also set the BPM.  
It's fine to use this to make videos, but please don't claim it as your own work.
